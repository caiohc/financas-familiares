# Controle Financeiro Familiar

Sistema avançado para controle interativo e inteligente de despesas e receitas usando *Clean Architecture*.

Esse projeto serve não somente como um aplicativo resolutivo na manutenção dos gastos familiares, como também é o um projeto base pedagógico destinado ao exercício e demonstração das melhores práticas e soluções técnicas presentes no ecossistema e ferramentas da linguagem Python.

## Consultando a Engenharia do Projeto

Em vez de centralizar tudo nesse documento, seguimos a prática de documentação granular.
Acesse o diretório `/docs` para mergulhar nos pormenores do projeto. Todo histórico mental está nas páginas a seguir:

- 📖 **[Requisitos Funcionais](docs/requisitos_funcionais.md)** - Funcionalidades da aplicação disponibilizadas para os usuários.
- ⚙️ **[Requisitos Não Funcionais](docs/requisitos_nao_funcionais.md)** - Necessidades, restrições técnicas e descrição da arquitetura do projeto.
